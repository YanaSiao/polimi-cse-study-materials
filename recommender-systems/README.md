## Study Suggestions

Based on past exams and the course structure:
- Focus on **definitions, properties, and key formulas** (types of recommenders, loss functions, system pipelines etc.).
- Be able to **compare algorithms**, not just describe them.
- Practice explaining **why** a specific algorithm fits a given scenario.
