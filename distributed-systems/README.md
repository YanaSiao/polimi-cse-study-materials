# Distributed Systems

This folder contains study materials for the **Distributed Systems** course, taken in the academic year **2025–2026**.

The course was taught by brilliant professors Cugola and Margara.


## 📂 Folder Contents

- `distributed_systems_questions.md`
  
  This file contains an **extraction of theory questions** from Distributed Systems exams, starting from **2014**.
  
  - Questions are **ordered by topic**
  - Useful for **targeted exam preparation**


- `suggestions.md`
  
  This file contains my **personal feedback and suggestions** for students approaching this course.
  
  It explains:
  - how to study Distributed Systems efficiently
  - how to use the questions file for preparation
  - strategies that worked for me to pass the exam

The suggestions are meant to **complement** the questions file and help reduce unnecessary trial-and-error during exam preparation.
